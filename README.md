# chakobsa-markov
Experiments in computational linguistics for the Chakobsa Dune scripts

# Script extracts

[`data/chakobsa.json`](data/chakobsa.json) is a JSON file containing the extracted Chakobsa words and phrases from the Dune scripts.

[`data/phonetic_dictionary.json`](data/phonetic_dictionary.json) is a JSON file containing the dictionary of Chakobsa words and phrases.
